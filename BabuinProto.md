# BabuinProto

Протокол обмена данными для мессенджера **Vexora**

---

⚠️ **Статус:** Проект находится на стадии разработки. Документация будет пополняться.

---

## Введение

BabuinProto — протокол, используемый для обмена данными между клиентом и сервером Vexora. Работает поверх JSON.

Каждый запрос содержит следующие поля:

| Поле | Тип | Описание |
| :---- | :---- | :---- |
| `method` | string | Категория операции |
| `operation` | string | Конкретное действие |
| `data` | any | Данные или аргументы, передаваемые в запрос |

**Пример структуры запроса:**

{

  "method": "authorisation",

  "operation": "register\_new\_user",

  "data": {“username”: "abobuss"}

}

---

## Методы (`methods`)

---

### `authorisation`

Содержит операции для регистрации, авторизации и проверки пользователей в базе данных.

---

#### `register_new_user`

Регистрирует нового пользователя в базе данных.

**Поле `data`:** `string` — юзернейм нового пользователя.

{

  "method": "authorisation",

  "operation": "register\_new\_user",

  "data": {“username”: "Nonelop"}

}

---

#### `check_user_username`

Проверяет, существует ли пользователь с указанным юзернеймом в базе данных.

**Поле `data`:** `string` — юзернейм для проверки.

{

  "method": "authorisation",

  "operation": "check\_user\_username",

  "data": {“username”: "Nonelop"}

}

---
